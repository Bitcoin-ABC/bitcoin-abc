# Connect

Connect to a Chronik node via `chronik-client`.

Import `ChronikClient` and provide it the Chronik URL in the constructor:

```tsx
import { ChronikClient } from 'chronik-client';
const chronik = new ChronikClient(['https://chronik-doc.fabien.cash']);
```

## Try it out!

You can try it out in this React component:

```jsx live
function DemoConnect() {
    return (
        <Json
            fn={async () => {
                const chronik = new ChronikClient([
                    'https://chronik-doc.fabien.cash',
                ]);
                return await chronik.blockchainInfo();
            }}
        />
    );
}
```
