# Bitcoin ABC 0.33.5 Release Notes

Bitcoin ABC version 0.33.5 is now available from:

  <https://download.bitcoinabc.org/0.33.5/>

This release includes the following features and fixes:
 - The `getpeerinfo` RPC now returns the time of the last message and the
   inflight bytes for each peer (fields `last_msg_start` and `bytes_inflight`).
